# Food-Sharing

## Idee
Der grösste Teil der Lebensmittelabfälle geschehen in der Schweiz im Haushalt. Mal zu viel gekocht, im Kühlschrank vergessen oder Fehlkäufe sind Gründe dafür. Deshalb habe ich eine Applikation entwickelt, mit der man im Quartier die überflüssigen Lebensmittel anbieten kann oder eine Lebensmittelanfrage erstellen kann. Dies für ein selbstbestimmter Preis. Jeder kann ein Quartier erstellen oder einem beitreten. 

## Circular Flow
- [ ] Use Longer
- [ ] Use Again
- [ ] Make Clean
- [X] Use Less

## Funktionen
- Anbieten von Lebensmittel inerhalb eines Quartiers
- Nachfragen von Lebensmittel innerhalb eines Quartiers
- Quartier erstellen
- Quartier beitreten

## Deployment
Diese App wird über Vercel veröffentlicht.
